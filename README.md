# SENTIMENT-ANALYSIS
# Sentiment Analysis using Titanic Dataset

## Project Overview
This project was completed as part of the CodeAlpha Data Analytics Internship. The aim of this task was to perform sentiment-style analysis using the Titanic dataset by categorizing survival outcomes into positive and negative sentiments.

---

## Objectives
- Perform sentiment-style classification
- Analyze passenger survival data
- Create visual representation of sentiments
- Understand distribution patterns

---

## Dataset Used
Titanic Dataset

---

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Analysis Performed
- Data loading and preprocessing
- Sentiment label creation
- Positive and negative classification
- Sentiment distribution analysis
- Gender vs sentiment visualization

---

## Files Included
- SentimentAnalysis.ipynb
- titanic.csv
- README.md

---

## Conclusion
The project demonstrated how classification-style sentiment analysis concepts can be applied to structured datasets using Python libraries and visual analytics.
